# account_management_java
A simple account management with Java

## Installation

### Database

In the package `xorm`, you will find an file name db.sql. Run this file or import it
on your database server.

It uses mysql


### Dependencies
AbsoluteLayout.jar

jcommon-1.0.23.jar

jfreechart-1.0.19

mysql-connector-java-8.0.22
